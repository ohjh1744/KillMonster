# KGA 유니티 미니 프로젝트 👥

## 게임 개요 🎮
### HuntBeast
- 3D FPS 게임
- 각 지역에서 등장한 괴물들을 무찌르고 현상금을 얻어 업그레이드하는 형식의 게임.
### Player 동작방식
- Player 동작방식
- 이동: W, A, S,D
- 달리기: 왼쪽 Shift
- 무기 교체: 1 : 주무기 , 2 : 보조무기
- 총 발사: 마우스 왼쪽 버튼
- 줌 : 마우스 오른쪽 버튼
- 칼로 베기: Z
- 수류탄 투척: T
- 재장전: R
### Boss
- 보스1: DemonKing
- 4가지 스킬 보유.
- Hp가 50이하로 떨어지면 분노하여 공격력 및 이동속도 상승.
- 분노 및 전역 스킬 사용 시 위험 문구 표시.
